# Main Features
- users List  
- Pagination with paging3
- user detail
- add favorite user to room DB
- navigation
- Network connection state with SnackBar

## Architecture
  - MVVM Architecture (Model - ComposableView - ViewModel)
  - Repository pattern
  - Hilt - dependency injection
